# 나이계산기 제작 대화 기록

> 날짜: 2026-06-02

---

## 1. 사용자 요청

> `@single-react-dev` 생년월일 입력하면 만 나이를 계산해주는 나이계산기를 만들어줘. 버튼을 누르면 계산이 되도록 만들고 `week-2\07-goblin` 폴더에 파일을 만들어줘

---

## 2. 작업 내용

`single-react-dev` 에이전트를 사용하여 생년월일을 입력하면 만 나이를 계산해주는 나이계산기를 제작했습니다.

### 파일 위치
`C:\Boot Camp\week-2\07-goblin\index.html`

### 기술 스택
- CDN 기반 **React + Tailwind CSS**
- 단일 HTML 파일 (별도 빌드 과정 없음)

### 핵심 기능

| 기능 | 설명 |
|------|------|
| 한국어 UI | 부드러운 그라데이션 배경의 중앙 정렬 카드 (반응형) |
| 생년월일 입력 + 버튼 | 날짜 입력 후 **"나이 계산하기"** 버튼을 눌렀을 때(또는 Enter)만 계산됨 |
| 만 나이 계산 | 올해 생일이 지났는지 여부로 한 살 차감, `new Date()`로 오늘 날짜를 동적 반영 |
| 결과 표시 | "만 OO세"로 강조 표시 + 다음 생일까지 남은 일수 표시 |
| 생일 당일 처리 | "오늘은 당신의 생일이에요!" 메시지 출력 |
| 입력 검증 | 빈 값 / 잘못된 날짜 / 미래 날짜에 대한 친절한 한국어 오류 메시지 |

### 구현 디테일
- `calculateAge` 함수: 현재 연도 − 출생 연도에서, 올해 생일이 아직 지나지 않았으면 1살 차감
- `new Date()`로 오늘 날짜를 동적으로 계산
- 날짜 입력의 `max`를 오늘로 제한
- 1900년 이전 날짜도 검증 처리

---

## 3. 사용 방법

브라우저로 바로 열어도 되지만, 로컬 서버 사용을 권장:

```powershell
npx serve "C:\Boot Camp\week-2\07-goblin"
```

또는 VS Code의 Live Server 확장으로 열기.

### 사용 흐름
1. 생년월일 입력
2. "나이 계산하기" 버튼 클릭
3. "만 OO세" 결과 및 다음 생일까지 남은 일수 확인
