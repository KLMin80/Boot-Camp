# 👥 경쟁 서비스 사용자 불만 분석 (AUDIENCES)

> 목적: **경쟁·인접 서비스 사용자들이 실제로 무엇에 불만인지**를 리뷰·앱스토어 별점·커뮤니티에서 모아, 톡트레이더가 어디를 파고들어야 하는지 도출한다. (research.md 후속)
> 조사일: 2026-07-11 · 방법: 앱스토어(구글플레이) 별점 분포·개발사 응답, Trustpilot/App Store/Reddit 후기, 국내 리뷰·나무위키 등 스크랩.
> ⚠️ 원문 개별 리뷰 전문은 플랫폼(구글플레이·Trustpilot)이 동적 로딩/차단하는 경우가 많아, **별점 분포 + 개발사 공개 응답(=사용자 불만을 역으로 드러냄) + 공개 후기 요약**을 근거로 정리했습니다.

---

## 요약: 사용자들은 결국 이 4가지에 화가 난다
세 서비스를 관통하는 공통 불만 패턴:
1. **💸 돈 관련 기습·부담** — 무료인 줄 알았는데 청구, 자산이 커질수록 커지는 수수료.
2. **🔒 블랙박스·경직성** — "왜 이렇게 매매됐는지" 모름, 내가 원하는 대로 못 바꿈.
3. **📉 백테스트/기대 ↔ 실전의 괴리** — 광고·시뮬레이션은 좋았는데 실계좌는 다름(슬리피지·손실).
4. **🙉 사람 없는 고객지원 / 돈 빼기 어려움** — 응답 없음, 출금 지연.

## 🇺🇸 Composer — "봇인 줄 알았더니 브로커, 그리고 청구서"
**별점/평판**: Trustpilot 다수 혼재 리뷰, Apple App Store 등. 호평(쉬운 시각적 UI, 슬랙 커뮤니티)과 강한 악평 공존.

**주요 불만 (별점 낮은 후기)**
- **기습 연회비 청구**: "무료 체험인 줄 알았는데 연 **$288 / $250 멤버십비**가 청구됐다." 항의하면 지원팀이 무성의하게 대응, 일부는 결제 분쟁 후 거래가 막혔다는 주장.
- **출금이 어렵고 고객지원이 사람이 아님**: 출금 방법 안내를 찾기 어렵고, 문의해도 응답 없이 대화가 사라짐. **AI-only 지원**이라 실제 상담 부재.
- **백테스트-실전 괴리(슬리피지)**: 거래량 적은 ETF에서 실전 슬리피지 발생, 시뮬 대비 성과 저하. "입금하자마자 **이틀 만에 -30%**", "$500 넣었는데 $319로".
- **비용 정당화 어려움**: 자동매매 **$32/월(또는 $24)** 은 최소 **$10k 이상** 굴려야 본전. 소액 사용자에겐 부담.
- **규제 혼란/SSN**: 트레이딩 봇으로 광고하지만 실제론 브로커라 **주민번호(SSN) 요구** → 오해·거부감.
- **기능 한도**: 전략 평가 도중 discovery/AI 사용 한도에 걸려 "정작 중요한 순간" 막힘.

## 🟡 핀트(Fint) — "맡겨놨더니 수익도 안 나고, 돈 빼기도 느리고"
**별점/평판(구글플레이)**: **4.5★** / 5★ 3,960 · 4★ 669 · 3★ 329 · 2★ 59 · **1★ 329개** → 별점은 높아 보여도 **1★가 3★와 동률**로 **양극화**(맡긴 만큼 실망도 큼).

**주요 불만 (개발사 응답이 역으로 드러내는 실제 항의)**
- **수익률 실망, 특히 단기**: 기대보다 낮은 수익/손실. (개발사 반복 응답: *"글로벌 자산배분·중위험중수익, 단기 대응 아닌 중장기"* → 이는 **단기 성과 불만**을 달래는 멘트.)
- **출금 지연**: *"펀드 특성상 **최대 17영업일**"*, *"오픈뱅킹 3일 이체제한"* 등 **돈 빼기가 느리다**는 항의.
- **경직성**: *"**1계좌=1계약** 구조라 한 계좌에서 여러 전략 운용 불가"* → 유연성 부족 불만.
- **완전 위임의 함정(투명성·수동성)**: "어디에 투자되는지 앱을 들여다봐야 안다", "AI가 다 해주니 **공부를 소홀히 하고 나태가 합리화**된다"는 자기비판적 후기.
- **수수료 + 원금손실**: 자산의 **연 0.768~1.176%를 매년** 부과, **비보호금융상품**이라 원금손실 가능, 성과수수료·배당세 별도.

## 🇰🇷 젠포트 — "백테스트는 화려한데 실전은 다르고, 전략이 새나갈까 불안"
**별점/평판(구글플레이)**: **4.7★** / 219개 중 5★ 171 · 4★ 33 · 3★ 6 · 2★ 0 · 1★ 4 → 대체로 호평이나 **뾰족한 페인포인트** 존재.

**주요 불만 (블로그·개발자 커뮤니티 후기)**
- **백테스트-실전 괴리**: *"실전 포트로 돌리면 결국 알파가 줄어든다"*, 추천 수익률과 실제 계좌 수익률의 괴리(슬리피지·체결). 젠트레이더를 병행해야 괴리 최소화.
- **보안·전략 탈취 우려**: **2020년 해킹 이력**, 요청 구조상 *"힘들게 만든 나만의 알고리즘이 남에게 쉽게 탈취될 수 있다"* 는 지적.
- **진입장벽/전문성**: "완벽하지 않은 백테스팅 플랫폼", 초보에겐 지표·조건식 학습이 여전히 벽.
- **구독료 대비 성과**: 실전을 제대로 쓰려면 상위 요금제 필요 → 성과가 비용을 정당화하는지에 대한 회의.

## 🎯 이 불만들이 톡트레이더에게 주는 시사점 (핵심)

| 경쟁자 사용자 불만 | 톡트레이더의 대응 | 판정 |
|---|---|---|
| 기습 연회비·월 구독료·AUM 수수료 | 내가 만든 개인용 → **구독료·운용수수료 0원** | ✅ 강점 |
| 블랙박스(핀트) / 플랫폼 틀에 갇힘(젠포트·Composer) | **손절선·수량·안전장치까지 100% 내 코드**, 매 주문 이유를 로그·톡으로 | ✅ 강점 |
| 경직성(핀트 1계좌=1계약, 전략 변경 제한) | 전략을 자유롭게 켜고/끄고/수정 | ✅ 강점 |
| 완전 위임의 투명성 부재 | 텔레그램으로 실시간 체결·사유 통보 | ✅ 강점 |
| 전략 탈취 우려(젠포트) | 전략이 **내 PC·내 코드에만** 존재 | ✅ 강점 (단 **API키·`.env` 보안 철저**) |
| 사람 없는 고객지원·출금 지연 | 내 계좌 직접 제어 → 출금은 증권사 통상, 지원 이슈 없음 | ✅ 강점 (단 **유지보수는 내 몫**) |
| **백테스트/시뮬 ↔ 실전 괴리, 슬리피지, 이틀 만에 -30%** | — | ⚠️ **나도 똑같이 겪을 수 있는 위험** |

### ⚠️ 남의 불만에서 배우는 "나도 조심할 것"
경쟁자 흉만 보면 안 됩니다. **가장 많이 반복된 진짜 위험은 "실전 괴리·급손실"** 입니다. 톡트레이더도 예외가 아니므로:
1. **슬리피지·체결 지연을 전제로 설계** — 시장가 주문 시 예상가와 체결가 차이를 로그로 남기고, 저유동 종목은 화이트리스트/한도로 제한. (MISSION 안전장치와 연결)
2. **"모의 검증 → 실전 전환 게이트"를 절대 건너뛰지 않기** — Composer 사용자의 "입금 직후 -30%"가 남 일이 아니다.
3. **API 키·전략 파일 보안** — 젠포트 해킹·전략 탈취 교훈: `.env`는 절대 커밋 금지, 키 유출 스캔.
4. **기대치 관리** — 핀트 1★의 본질은 "기대보다 낮은 수익". 톡트레이더의 목표도 "대박"이 아니라 **손실 통제 기반의 안정적 구조**임을 스스로 명확히.

## 🧑‍🤝‍🧑 정리: 톡트레이더가 노리는 사용자(=나)
- 경쟁 서비스의 **구독료·수수료가 아깝고**, **AI에 다 맡기긴 불안하며(왜 샀는지 알고 싶고)**, **내 규칙대로 바꾸고 싶은** 개인 투자자.
- 이들이 겪는 불만(비용·블랙박스·경직성)을 정확히 뒤집는 것이 톡트레이더의 자리. 대신 **실전 괴리·급손실·보안**이라는 공통 위험은 내가 직접 짊어지므로, MISSION의 안전 원칙이 곧 경쟁력이다.
