# 🧺 남김없이

## 사놓고 잊어버린 재료, 이번엔 **남김없이** 다 쓰세요

냉장고 뒤에서 곤죽이 된 채소, 유통기한 지난 두부, 언제 샀는지 기억도 안 나는 소스들…
매달 이렇게 버리는 식재료가 **3~4만 원어치**예요. '깜빡해서'가 아니라, 관리가 귀찮아서죠.

**남김없이**는 그 귀찮음을 대신 해드려요.
**장 본 영수증 한 장으로 담고 · 상하기 전에 알려주고 · 그 재료로 뭘 해 먹을지까지** 이어줍니다.

> 버리는 음식이 줄고, 새어나가던 식비가 눈에 보이게 줄어들어요.

## 보통 냉장고 앱과 뭐가 다를까요?

대부분의 냉장고 앱은 **유통기한 알림**에서 멈춰요.
남김없이는 **장 보는 순간부터 다 쓸 때까지** 전부를 이어서 도와줘요.

| | 보통 냉장고 앱 | 🧺 남김없이 |
|---|---|---|
| **담기** | 품목을 하나씩 손으로 입력 | 🧾 **영수증 한 장**이면 품목·가격이 한꺼번에 (+ 라벨 사진 자동 인식) |
| **알림** | 유통기한만 알려주고 끝 | 📉 **상하기 전** + 소비 속도로 **떨어지기 전**까지, 둘 다 미리 |
| **레시피** | 없거나 따로 검색 | 🍲 지금 재료로 추천 → **만들면 그 재료가 자동으로 차감** |
| **주문** | 있어도 한 쇼핑몰에 묶임 | 🛒 **쿠팡·컬리·이마트몰·네이버**를 나란히 놓고 비교 |
| **절감** | 얼마 아꼈는지 알 수 없음 | 📊 **아낀 돈이 숫자로** 보이는 리포트 |

> 한마디로 — 다른 앱은 "언제 상하는지" 알려주고, 남김없이는 **"어떻게 다 쓸지"**까지 함께해요.

## 이렇게 좋아져요

- 🧾 **영수증만 찍으면 한꺼번에** — 장 본 영수증 한 장으로 품목·가격을 다 담아요 (유통기한만 확인하면 끝)
- 📷 **라벨은 사진으로 끝** — 제품명·용량·유통기한을 알아서 읽어 담아줘요
- 🔴 **버리기 전에 알려줘요** — 유통기한이 다가오면 빨갛게, 여유 있으면 초록으로 한눈에
- 📉 **떨어지기 전에 미리** — 쓰는 속도를 보고 "곧 떨어져요"를 알려줘 미리 담게 해줘요
- 🧊 **오늘 못 쓰겠으면 냉동실로** — 버튼 하나로 기한 연장 (얼리면 안 되는 건 막아줘요)
- 🍲 **"오늘 뭐 먹지?"를 대신 고민** — 지금 재료로 급한 것부터 추천, **오늘 쓰고 싶은 재료를 골라서** 받을 수도 있어요
- ✏️ **쓴 만큼만 차감** — 앱 없이 그냥 해 먹어도 "조금 썼어요"로 남은 양을 맞춰요
- 🛒 **부족한 재료는 바로 비교 주문** — 쿠팡·컬리·이마트몰·네이버를 나란히 놓고 골라요
- 📊 **버린 돈이 줄어드는 게 보여요** — 이번 달 얼마나 아꼈는지 한눈에

## 처음이신가요? 5분이면 익혀요

### 1️⃣ 시작하기
앱 주소를 열고 **이메일·비밀번호로 가입**하세요. 끝나면 바로 내 냉장고가 열려요.
> 💡 크롬 메뉴에서 **‘홈 화면에 추가’** 하면 앱처럼 아이콘으로 열 수 있어요.

### 2️⃣ 장 본 걸 담기 — 하단 **‘담기’**
- 🧾 **영수증으로 한 번에** (제일 편해요) — 장 보고 온 영수증을 찍거나 갤러리에서 고르면, **품목·가격을 한꺼번에** 담아요. 봉투·할인 같은 건 알아서 빼줘요.
- 📷 **사진으로 담기** — 우유·두부·고기처럼 라벨 있는 건 라벨만 가까이 찍으면 제품명·용량·유통기한을 읽어와요.
- 🥬 **재료로 담기** — 양파·사과처럼 라벨 없는 건, 재료를 고르고 **어디에 둘지**(냉장·냉동·실온·그늘)만 누르면 유통기한이 자동으로 잡혀요.
- 🍗 **남은 음식**(김치, 어제 먹다 남은 치킨, 명절 전·잡채 등)도 담아두면 레시피에 활용돼요.

### 3️⃣ 확인하기 — 하단 **‘확인’**
정리 다 끝내고 편히 앉아서, 읽어온 값이 맞는지 카드로 넘기며 확인하세요.
**영수증엔 유통기한이 없으니**, 여기서 날짜만 사진이나 손으로 채우면 돼요. 틀린 값은 **‘고칠게요’**로 그 자리에서 수정(용량·단위·가격·날짜까지).

### 4️⃣ 냉장고 보기 — 하단 **‘냉장고’**
- **색이 곧 남은 시간**이에요 — 빨강(급함) → 주황(이번 주) → 초록(여유).
- 위에 **‘곧 떨어져요’** 카드가 뜨면, 쓰는 속도상 곧 바닥나는 재료예요. **‘미리 담기’**로 바로 비교 주문.
- 재료를 눌러 **‘조금 썼어요’**(¼·½·¾·직접) 로 쓴 만큼만 빼거나, **‘냉동실로’** 로 기한을 늘려요.

### 5️⃣ 오늘 뭐 먹지 — 하단 **‘요리’**
- 지금 재료로 만들 요리를 **급한 재료부터** 보여줘요. 눌러서 **만드는 법**을 보고, **몇 인분**인지 고르면 재료 양이 맞춰져요.
- 🧺 **‘냉장고에서 골라 레시피 만들기’** — 오늘 꼭 쓰고 싶은 재료를 냉장고에서 여러 개 고르거나 **직접 입력**(예: 어제 남은 치킨)하면, 그걸 꼭 쓰는 레시피를 새로 만들어줘요.
- 🍳 실제로 만들었으면 **‘다 만들었어요 · 재료 빼기’** — 그 레시피 재료가 냉장고에서 자동으로 차감돼요.

### 6️⃣ 부족한 재료 주문하기
레시피에 부족한 재료가 있으면 **쿠팡·컬리·이마트몰·네이버를 비교**해 바로 주문해요.
미리 사뒀다면 **‘주문했어요’**를 눌러 ‘곧 도착’으로 챙겨두면, 그 재료까지 넣어 레시피를 미리 보여드려요.

### 7️⃣ 아낀 돈 확인 — 하단 **‘리포트’**
다 먹고·버린 기록이 쌓여서, **이번 달 버린 돈과 줄어든 추세**로 보여요. 가격을 넣을수록 정확해져요.

## 알아두면 좋아요

- 🖼 **갤러리에서 안 열리면** — 카카오톡 같은 앱 안 브라우저 대신 **크롬으로 열어주세요**. (⋮ 메뉴 → 다른 브라우저로 열기)
- 🔙 **뒤로가기**를 눌러도 앱이 꺼지지 않고 이전 화면으로 돌아가요.
- 🧊 두부·감자처럼 **얼리면 안 되는 재료**는 냉동 버튼이 알아서 막아줘요.
- 📷 사진이 잘 안 읽히면 — **글자가 화면을 채우게 가까이**, 빛 반사 없이 또렷하게. (연도가 없어도 앱이 알아서 채워요)
