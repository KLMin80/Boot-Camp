# 📱 앱스토어 업로드 가이드 — 남김없이 (안드로이드)

배포된 웹앱(`nowaste-app-seven.vercel.app`)을 **Capacitor로 감싸** 안드로이드 앱으로 만들고 **구글 Play**에 올리는 절차예요.
아이팟/아이폰(iOS)은 **맥이 있어야** 해서 이번엔 안드로이드 먼저.

> 이 PC엔 JDK·안드로이드 SDK가 없어서 여기서 빌드는 못 해요. **당신 PC에 Android Studio를 깔고** 아래대로 하면 됩니다. 코드·설정·아이콘·문구는 다 준비해뒀어요(`store-assets/`).

---

## 0) 당신이 준비할 것
- [ ] **Android Studio** 설치 (JDK 포함됨) — [developer.android.com/studio](https://developer.android.com/studio)
- [ ] **Node.js** (이미 있음)
- [ ] **구글 Play 개발자 계정** — [play.google.com/console](https://play.google.com/console) · 최초 **$25** 1회 결제 + 본인확인
- [ ] **서명 키(keystore)** — 빌드 때 만들고 **절대 분실 금지**(잃으면 앱 업데이트 영영 불가)

---

## 1) Capacitor 프로젝트 만들기 (당신 PC 터미널)
아무 새 폴더에서:
```bash
mkdir nowaste-android && cd nowaste-android
npm init -y
npm i @capacitor/core @capacitor/cli @capacitor/android
npx cap init "남김없이" com.nowaste.app --web-dir=www

# 원격 배포 URL을 로드하도록 설정 (준비해둔 파일 사용)
#  → 이 저장소의 store-assets/capacitor.config.json 내용을 복사해 붙여넣기
#    (핵심: "server": { "url": "https://nowaste-app-seven.vercel.app" })

# webDir 폴백용 최소 파일
mkdir www && echo "<meta http-equiv=refresh content='0; url=https://nowaste-app-seven.vercel.app'>" > www/index.html

npx cap add android
npx cap sync
npx cap open android      # Android Studio가 열림
```
> **왜 원격 URL을 로드?** 이 앱은 프론트+서버(로그인·DB·OCR)가 한 몸이라, 앱 껍데기가 **배포된 사이트를 그대로 띄우는** 방식이 가장 간단하고 안전해요. 카메라·갤러리도 웹뷰에서 그대로 됩니다. 앱을 고칠 땐 웹만 재배포하면 앱도 같이 최신이 돼요.

## 2) 아이콘 넣기
- 준비된 마스터: **`store-assets/app-icon-1024.png`** (🧺)
- Android Studio → `res` 우클릭 → **New → Image Asset** → Foreground에 이 파일 → 밀도별 아이콘 자동 생성
- (또는) `npm i @capacitor/assets -D` 후 `npx capacitor-assets generate` 로 일괄 생성

## 3) 권한 (카메라)
`android/app/src/main/AndroidManifest.xml` 에 카메라 사용 명시(사진 촬영용):
```xml
<uses-permission android:name="android.permission.CAMERA"/>
<uses-feature android:name="android.hardware.camera" android:required="false"/>
```
인터넷 권한은 기본 포함. (웹뷰가 HTTPS 원격을 로드하므로 cleartext 불필요)

## 4) 서명된 AAB 빌드
Android Studio → **Build → Generate Signed Bundle / APK → Android App Bundle(AAB)**
- 처음이면 **Create new keystore** → 비밀번호·별칭 정하고 **파일 안전히 백업**(분실 = 업데이트 불가)
- 릴리스 빌드 → `app-release.aab` 생성 → 이걸 Play Console에 업로드

---

## 5) Play Console 제출
1. **앱 만들기** — 이름 `남김없이`, 무료, 패키지명 `com.nowaste.app`
2. **개인정보처리방침 URL** → `https://nowaste-app-seven.vercel.app/privacy` ✅ *(이미 준비됨)*
3. **데이터 안전(Data safety)** 설문 — privacy.html대로 신고:
   - 수집: 이메일(계정), 사진(앱 기능·OCR), 앱 활동(냉장고 데이터)
   - 제3자 공유: OpenAI(사진→텍스트 추출) — "처리 위탁"
   - 암호화 전송(HTTPS) ✅, 삭제 요청 가능 ✅
4. **콘텐츠 등급** 설문(전체이용가 예상), **대상 연령**(만 14세 이상 권장 — privacy.html과 일치)
5. **스토어 등록정보** — 아래 4·6·7절 문구/자산 사용
6. **출시 트랙**: 내부 테스트 → (선택)비공개 테스트 → 프로덕션. 처음엔 **내부 테스트**로 본인 기기에서 확인 추천

## 6) 스토어 등록정보 (복사해서 쓰기)

**앱 이름:** 남김없이
**짧은 설명(80자):**
> 영수증 한 장으로 담고, 상하기 전에 알려주고, 다 쓰게 — 음식물 쓰레기·식비 줄이는 냉장고 앱

**자세한 설명:**
```
사놓고 잊어버린 재료, 이번엔 남김없이 다 쓰세요.

보통 냉장고 앱은 '유통기한 알림'에서 멈춥니다.
남김없이는 장 보는 순간부터 다 쓸 때까지 함께해요.

• 🧾 영수증 한 장이면 품목·가격이 한꺼번에 (라벨은 사진으로 자동 인식)
• 🔴 상하기 전에 — 색이 곧 남은 시간 (급함은 빨강, 여유는 초록)
• 📉 떨어지기 전에 — 쓰는 속도로 '곧 떨어져요'를 미리 알림
• 🍲 오늘 뭐 먹지 — 지금 재료로 레시피 추천, 만들면 재료가 자동 차감
• ✏️ 쓴 만큼만 차감 — 앱 없이 해 먹어도 남은 양을 맞춤
• 🧊 냉동실로 구제 — 오늘 못 쓰겠으면 버튼 하나로 기한 연장
• 🛒 비교해서 주문 — 쿠팡·컬리·이마트몰·네이버를 나란히
• 📊 아낀 돈이 보여요 — 이번 달 얼마나 덜 버렸는지 리포트

버리는 음식이 줄고, 새어나가던 식비가 눈에 보이게 줄어듭니다.
```
**카테고리:** 음식/음료 (또는 라이프스타일)
**연락처 이메일:** [운영자 이메일 입력]
