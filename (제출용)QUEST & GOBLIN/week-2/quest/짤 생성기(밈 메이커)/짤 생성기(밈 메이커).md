# 짤 생성기 (밈 메이커) 개발 대화 기록

> 작성일: 2026-06-05
> 결과물: `index.html` (단일 파일 React 앱)

---

## 1차 요청 — 밈 메이커 기본 제작

**사용자**
> 짤 생성기(밈 메이커)를 만들어줘. 이미지 위에 텍스트를 배치해주고 Canvas로 다운 가능한 이미지로 생성해줘.
> `week-2\quest\짤 생성기(밈 메이커)` 폴더에 파일로 만들어줘.

**작업 내용**
- CDN 기반 React 18 + Babel + Tailwind CSS로 단일 `index.html` 제작
- `single-react-dev` 에이전트를 사용해 제작

**구현된 기능**
1. **이미지 업로드** — 파일 선택 버튼 + 드래그앤드롭, 비어있을 때 드롭존 placeholder
2. **샘플 템플릿** — CORS 안전한 SVG 배경 4종 + 온라인 밈 템플릿(드레이크/도지, `crossOrigin` 처리)
3. **텍스트 오버레이** — 기본 위/아래 텍스트, `＋ 텍스트 추가`로 무제한 블록 추가
4. **블록별 조절** — 내용·글자 크기·글자색·외곽선 색·외곽선 두께·대문자 토글·위치(가로/세로)
5. **드래그 이동** — 캔버스 위에서 마우스/터치로 글자를 직접 끌어 이동 (표시/해상도 스케일 보정)
6. **Canvas 렌더링** — 실제 `<canvas>`가 단일 소스, Impact 폰트 + 흰 글자/검은 외곽선 클래식 밈 스타일, 자동 줄바꿈
7. **PNG 다운로드** — `canvas.toBlob` → `meme.png`, CORS 오염 시 한국어 에러 폴백

**핵심 설계**
- 텍스트 위치를 0~1 비율(`xRatio`/`yRatio`)로 저장 → 이미지 해상도가 바뀌어도 안정적
- 슬라이더와 드래그가 동일 상태를 공유

## 2차 요청 — 효과(애니메이션) 옵션 추가

**사용자**
> 밈을 만들 때 옵션으로 선택해서 이미지 파일에 간단한 효과(떨림, 반으로 갈라짐, Fade in&out 등)를 넣고 싶은데 이러한 기능을 선택할 수 있도록 반영해줄 수 있니?

**핵심 판단**
- 효과는 애니메이션이므로 정적 PNG로 담을 수 없음 → 실시간 애니메이션 미리보기 + **움직이는 GIF("짤") 내보내기**로 확장

**추가된 효과 (2번 카드: 효과 선택)**
| 효과 | 설명 |
|------|------|
| ⬜ 없음 | 정지 이미지 (PNG) |
| 📳 떨림 | 이미지가 주기적으로 부르르 떨림 (가장자리 빈틈 방지 위해 살짝 확대 렌더) |
| ✂️ 반으로 갈라짐 | 좌/우 절반이 벌어졌다 붙음 |
| 🌗 페이드 인&아웃 | 서서히 나타났다 사라짐 |

- 효과별 **강도**(떨림·갈라짐), **속도**, **배경색**(갈라짐·페이드 빈 영역) 슬라이더 제공

**동작 방식**
- **실시간 미리보기**: `requestAnimationFrame` 루프로 효과를 입혀 표시 ("▶ 애니메이션 적용 중" 배지)
- **내보내기 자동 분기**
  - 효과 없음 → `meme.png`
  - 효과 있음 → 움직이는 `meme.gif` (`gif.js` 인코딩, 진행률 % + 프로그레스바, 무한 반복)
  - 애니메이션 모드에서도 "정지 이미지(PNG)로도 저장" 버튼 별도 제공
- 그리기 로직을 `composeMeme`(이미지+텍스트 합성) / `drawMeme`(효과+진행도 렌더)로 분리
  → **미리보기와 GIF 인코딩이 동일 렌더러**를 사용해 WYSIWYG 보장

**기술 디테일**
- GIF 용량 절약을 위해 최대 480px로 축소, 18프레임 인코딩
- `gif.worker.js`를 blob URL로 받아 교차출처 worker 이슈 회피 (실패 시 CDN URL 폴백)
- 효과 루프는 사인/코사인 기반이라 시작·끝 프레임이 매끄럽게 이어짐
- CORS로 오염된 온라인 템플릿은 GIF 인코딩 시 친절한 한국어 에러로 폴백

---

## 사용된 기술 스택

- **React 18** (CDN, `react.development.js` / `react-dom.development.js`)
- **Babel Standalone** (`type="text/babel"`로 JSX 변환)
- **Tailwind CSS** (CDN)
- **gif.js** `0.2.0` (움직이는 GIF 인코딩)
- 빌드 도구 없음 — 단일 `index.html` 파일

---

## 실행 방법

- VS Code **Live Server** 확장으로 `index.html` 열기, 또는
- 해당 폴더에서 터미널에 `npx serve .` 실행 후 표시된 주소 접속
- (더블클릭으로도 동작하나, GIF worker 로딩과 온라인 템플릿 CORS 처리를 위해 로컬 서버 사용 권장)

## 파일 구조 (`index.html` 내부)

```
<head>
  ├─ Tailwind / React / ReactDOM / Babel / gif.js (CDN)
  └─ <style> 드래그·드롭존 애니메이션 CSS

<script type="text/babel">
  ├─ 🎨 Design System: Button / Card / Input / Slider / ColorPicker / Toggle
  ├─ 🧩 상수: FONT_STACK, EFFECTS, 샘플/온라인 템플릿, defaultTextBlock
  ├─ 🧰 렌더링 유틸: wrapLines / drawTexts / composeMeme / drawMeme
  ├─ 📄 TextBlockEditor (텍스트 블록 편집기)
  ├─ 📄 EffectPanel (효과 선택)
  ├─ 📄 MemeCanvas (드래그 가능한 캔버스)
  ├─ 📄 DropZone (이미지 없을 때)
  └─ 🚀 App (전체 조합 + 상태 관리)
```
