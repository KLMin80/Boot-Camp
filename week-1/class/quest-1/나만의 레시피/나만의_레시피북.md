## 1. 김치찌개

🍲 **분량** 2인분 | ⏱️ **조리시간** 25분 | 🌶️ **난이도** ★★☆☆☆

### 재료
- 잘 익은 신김치 1/4포기 (약 300g)
- 돼지고기 (앞다리살·목살) 200g
- 두부 1/2모
- 대파 1대
- 양파 1/2개
- 김치국물 3큰술
- 물 (또는 쌀뜨물) 2.5컵

**양념**
- 고춧가루 1큰술
- 다진 마늘 1큰술
- 국간장 1큰술
- 설탕 1/2작은술
- 후추 약간

### 🔄 조리 흐름 한눈에 보기

```
🔪 재료 썰기  →  🥩 고기·김치 볶기  →  🌶️ 양념 볶기
                                          ↓
🥣 완성!  ←  🧈 두부·대파 + 간  ←  💧 물 붓고 끓이기 → 🧅 양파 넣고 푹
```

🔪 → 🥩 → 🌶️ → 💧 → 🧅 → 🧈 → 🥣

### 만드는 법
1. 🔪 신김치는 한입 크기로 썰고, 돼지고기는 먹기 좋게 썬다.
2. 🥩 냄비에 기름을 약간 두르고 돼지고기를 중불에서 볶다가 김치를 넣어 5분간 함께 볶는다.
3. 🌶️ 고춧가루·다진 마늘을 넣고 1분 더 볶아 양념을 입힌다.
4. 💧 물(쌀뜨물)과 김치국물을 붓고 센 불로 끓인다.
5. 🧅 끓어오르면 양파를 넣고 중불로 줄여 15분간 푹 끓인다.
6. 🥣 두부와 대파를 넣고 국간장·설탕·후추로 간을 맞춰 5분 더 끓이면 완성.

> 💡 **TIP** 쌀뜨물을 쓰면 국물이 더 구수하고, 김치를 미리 볶아주면 깊은 맛이 살아납니다.

## 2. 제육볶음

🥘 **분량** 2인분 | ⏱️ **조리시간** 20분 (+재움 30분) | 🌶️ **난이도** ★★☆☆☆

### 재료
- 돼지고기 앞다리살 (또는 목살) 400g
- 양파 1개
- 대파 1대
- 양배추 1/4통 (선택)
- 청양고추 1~2개

**양념장**
- 고추장 2큰술
- 고춧가루 2큰술
- 간장 2큰술
- 다진 마늘 1.5큰술
- 설탕 1큰술 (또는 물엿 1.5큰술)
- 맛술 1큰술
- 참기름 1큰술
- 후추·생강가루 약간

### 🔄 조리 흐름 한눈에 보기

```
🥣 양념장 섞기  →  ⏲️ 고기 재우기(30분)  →  🧅 양파 먼저 볶기
                                              ↓
✨ 참기름 마무리  ←  🥬 채소 넣고 볶기  ←  🔥 고기 센 불에 볶기
```

🥣 → ⏲️ → 🧅 → 🔥 → 🥬 → ✨

### 만드는 법
1. 🥣 양념장 재료를 모두 섞어 둔다.
2. ⏲️ 돼지고기에 양념장을 넣고 버무려 30분 이상 재운다. (시간 없으면 10분도 OK)
3. 🧅 달군 팬에 식용유를 두르고 양파를 먼저 볶는다.
4. 🔥 재워둔 고기를 넣고 센 불에서 볶는다.
5. 🥬 고기가 거의 익으면 양배추·대파·청양고추를 넣고 빠르게 볶는다.
6. ✨ 고기가 완전히 익고 양념이 졸아들면 참기름을 둘러 마무리.

> 💡 **TIP** 센 불에서 빠르게 볶아야 불맛이 나고 고기가 질겨지지 않습니다. 상추쌈과 함께 드세요!

## 3. 중국식 볶음밥

🍚 **분량** 2인분 | ⏱️ **조리시간** 15분 | 🌶️ **난이도** ★★☆☆☆

### 재료
- 찬밥 2공기 (고슬고슬한 밥일수록 좋음)
- 달걀 2개
- 대파 1대 (송송 썰기)
- 다진 마늘 1작은술
- 햄 또는 새우·게맛살 1/2컵
- 당근·양파 약간 (잘게 깍둑썰기)
- 식용유 2큰술

**양념**
- 굴소스 1큰술
- 간장 1큰술
- 소금·후추 약간
- 참기름 1/2큰술

### 🔄 조리 흐름 한눈에 보기

```
🍳 달걀 스크램블  →  🧄 파기름 내기  →  🥕 채소·햄 볶기
                                          ↓
✨ 달걀 + 참기름 마무리  ←  🥢 간장·굴소스 간  ←  🍚 찬밥 넣고 볶기
```

🍳 → 🧄 → 🥕 → 🍚 → 🥢 → ✨

### 만드는 법
1. 🍳 팬에 식용유를 넉넉히 두르고 달걀을 풀어 스크램블한 뒤 따로 덜어 둔다.
2. 🧄 같은 팬에 기름을 두르고 다진 마늘·대파를 넣어 파기름을 낸다.
3. 🥕 당근·양파·햄을 넣고 센 불에서 볶는다.
4. 🍚 찬밥을 넣고 주걱으로 눌러가며 밥알이 흩어지도록 볶는다.
5. 🥢 간장은 팬 가장자리에 둘러 불향을 내고, 굴소스·소금·후추로 간한다.
6. ✨ 스크램블 달걀을 다시 넣고 골고루 섞은 뒤 참기름을 둘러 마무리.

> 💡 **TIP** 반드시 **찬밥**을 사용하고 센 불에서 빠르게 볶아야 고슬고슬해집니다. 간장을 팬 가장자리에 부으면 불맛(웍헤이)이 살아나요.

## 4. 그릭요거트 과일 볼 (아침 5분 간편 건강식)

🥣 **분량** 1인분 | ⏱️ **조리시간** 5분 | 🌶️ **난이도** ★☆☆☆☆

> 불 없이 섞기만 하면 완성! 단백질·식이섬유·비타민을 한 번에 채우는 든든한 아침.

### 재료
- 그릭요거트 (무가당) 150g
- 바나나 1/2개
- 블루베리·딸기 등 좋아하는 과일 한 줌
- 견과류 (아몬드·호두) 1줌
- 오트밀(또는 그래놀라) 2큰술
- 꿀 또는 메이플시럽 1작은술
- 치아씨드 1작은술 (선택)

### 🔄 조리 흐름 한눈에 보기

```
🥣 요거트 담기  →  🍌 과일 올리기  →  🌰 오트밀·견과류
                                          ↓
                        🥄 완성!  ←  🍯 꿀 두르기
```

🥣 → 🍌 → 🌰 → 🍯 → 🥄

### 만드는 법
1. 🥣 그릇에 그릭요거트를 담는다.
2. 🍌 바나나는 동그랗게 썰고, 과일은 한입 크기로 손질해 올린다.
3. 🌰 오트밀(그래놀라)과 견과류를 뿌린다.
4. 🍯 꿀이나 메이플시럽을 살짝 둘러 단맛을 더한다.
5. 🥄 치아씨드를 뿌리면 영양 가득! 바로 떠서 즐긴다.

> 💡 **TIP** 전날 밤 오트밀을 우유·요거트에 불려두면(오버나이트 오트밀) 더 부드럽고 포만감이 큽니다. 과일은 냉동 베리를 써도 좋아요.
