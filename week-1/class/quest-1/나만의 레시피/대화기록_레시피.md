## 🗨️ 1. 레시피 북 제작 요청

**👤 사용자**
> 나만의 레시피 북을 만드려고 해. 김치찌게, 제육볶음, 중국식 볶음밥 3가지 요리의 레시피와 아침에 5분만에 만들어 먹을 수 있는 간편 건강식 레시피 1가지를 추천해서 총 4가지 요리의 레시피를 레시피 북으로 만들 수 있도록 만들어줘

**🤖 Claude**
- 요청한 3가지 요리 + 아침 5분 건강식으로 **그릭요거트 과일 볼**을 추천하여 총 4가지 레시피 작성
- `나만의_레시피북.md` 파일 생성
- 구성: 목차(클릭 이동 링크), 레시피별 분량·조리시간·난이도, 재료/만드는 법/TIP, 마지막 요약 표

| 요리 | 분량 | 조리시간 |
|------|------|----------|
| 🍲 김치찌개 | 2인분 | 25분 |
| 🥘 제육볶음 | 2인분 | 20분 |
| 🍚 중국식 볶음밥 | 2인분 | 15분 |
| 🥣 그릭요거트 과일 볼 (아침 5분 건강식) | 1인분 | 5분 |

## 🗨️ 2. 인포그래픽 이미지 요청

**👤 사용자**
> 조리방법을 쉽게 이해할 수 있도록 인포그래픽이미지를 넣어줘

**🤖 Claude**
- 사진 같은 AI 생성 이미지(JPG/PNG)는 직접 만들 수 없음을 안내
- 대신 아이콘·번호·화살표로 단계를 시각화한 **HTML + SVG 인포그래픽 버전** 제안 → (사용자가 진행 중단)

---

## 🗨️ 3. 아이콘 + 화살표 표시 요청

**👤 사용자**
> 이미지를 생성할 수 없으면 아이콘 및 화살표표시는 넣을 수 있니?

**🤖 Claude**
- 가능! 이모지 아이콘 + 화살표(→)로 조리 흐름을 마크다운에 시각화
- 각 레시피에 **「🔄 조리 흐름 한눈에 보기」** 섹션 추가
  - 화살표 흐름 다이어그램 (ㄹ자 형태)
  - 한 줄 요약 흐름 라인
  - 단계별 본문에도 아이콘 매칭

| 레시피 | 흐름 |
|--------|------|
| 🍲 김치찌개 | 🔪 → 🥩 → 🌶️ → 💧 → 🧅 → 🧈 → 🥣 |
| 🥘 제육볶음 | 🥣 → ⏲️ → 🧅 → 🔥 → 🥬 → ✨ |
| 🍚 중국식 볶음밥 | 🍳 → 🧄 → 🥕 → 🍚 → 🥢 → ✨ |
| 🥣 그릭요거트 볼 | 🥣 → 🍌 → 🌰 → 🍯 → 🥄 |

## 🗨️ 5. 대화 캡쳐 요청

**👤 사용자**
> 지금까지 너와 대화한 내용을 캡쳐할 수 있니? 할 수 있다면 캡쳐해서 week-1\quest-1 폴더에 넣어줘.

**🤖 Claude**
- 화면 스크린샷(이미지) 캡쳐는 불가능함을 안내 (채팅 화면에 접근 불가)
- 직접 캡쳐 방법 안내: `Win + Shift + S`
- 대안으로 대화 내용을 글로 정리한 기록 파일 제안 → 사용자가 1번(대화 기록 마크다운) 선택
