# MARKETPLACES.md — 멀티 마켓 & 제휴 수익

> 여러 마켓으로 보내 사용자가 직접 비교하게 한다. **쿠팡이 구조적으로 못 하는 자리(중립 비교)를 차지하는 게 이 앱의 유일한 해자** ([STRATEGY.md](STRATEGY.md)).
> 조사일 2026-07. 출처는 문서 하단.

## 1. 제휴로 나에게 돈이 되는가 (검증)

| 마켓 | 제휴 프로그램 | 수수료 | 확인도 | v1 |
|------|--------------|--------|--------|----|
| **쿠팡** | 쿠팡파트너스(직접) | ~3% | ✅ 확실 | 딥링크(승인 후 파트너스 링크로 교체) |
| **네이버** | **쇼핑커넥트**(2025 출시) | 상품별 | ✅ 확실 · **쿠팡파트너스와 동시 운영 가능** | 딥링크 + 참고가 |
| **11번가·G마켓** | 링크프라이스·애드픽 등 CPS망 | ~3~5% | ✅ 가능(링크프라이스는 트래픽 있어야 승인) | 트래픽 쌓인 뒤 추가 |
| **마켓컬리·SSG/이마트몰** | 공개 제휴 **미확인** | — | ⚠️ 직접 문의 필요 | 딥링크(수익 없이 먼저, 확인되면 켬) |

**당장 확실히 돈이 되는 것: 쿠팡파트너스 + 네이버 쇼핑커넥트** (둘은 공존 가능). 식료품 전문인 **컬리·SSG는 공개 제휴가 확인 안 됨** → 직접 문의 대상.

## 2. 이용자 규모 (2025 MAU) — 어느 걸 넣을 가치가 있나

쿠팡 3,430만(압도) · 알리 920만 · 테무 830만 · 11번가 790만 · G마켓 680만 · 네이버플러스 510만(성장 1위).
- 알리·테무는 신선식품 아님 → 제외. 올리브영·에이블리·무신사·다이소도 비식료품 → 제외.
- **식료품 기준 실속: 쿠팡 · 네이버 · (일반+식료품)11번가·G마켓 · (식료품 전문)컬리·SSG.**
- v1은 **쿠팡·컬리·SSG·네이버 4개**로 시작(식료품 밀착). 11번가·G마켓은 트래픽 후 추가.

## 3. ⚠️ 왜 "실시간 가격 비교"는 불가한가 (담벼락)

가격을 나란히 놓으려면 각 마켓 가격 데이터가 필요한데 두 거인이 담을 쳤다:
- **쿠팡**: 상품 가격을 읽는 공개 API 없음. 파트너스는 **딥링크만** 준다.
- **네이버 쇼핑 검색 API**(`lprice`·`mallName`·`link`, 무료)는 가격비교 소스지만 **쿠팡·G마켓·11번가가 배제됨**(네이버가 쇼핑앱서 가격비교 제거).
- 스크래핑은 약관 위반 + 매일 깨짐 → 기각.

→ **결론: "쿠팡 3,000 vs 컬리 3,500" 실시간 나란히 비교는 기술적으로 막혀 있다.**

## 4. 그래서 만든 것 (구현됨)

`server.js`의 `/api/buy-links` — 재료마다 **4개 마켓 검색 딥링크** + (네이버 키 있을 때) **참고 최저가**.

```
부족 재료: 돼지고기
  🚀 쿠팡프레시   → 딥링크 (파트너스 수익)
  🥬 마켓컬리     → 딥링크 (제휴 확인 시 수익)
  🏪 이마트몰     → 딥링크 (제휴 확인 시 수익)
  🔎 네이버쇼핑   → 딥링크 (쇼핑커넥트 수익) + "최저 2,890원~"(키 있을 때)
```

- 사용자는 탭 한 번으로 마켓 이동·비교. 어느 걸 눌러도(제휴된 것은) 수익.
- **URL은 `MARKETS` 배열 한 곳에서만 생성** → 제휴 승인되면 그 함수만 딥링크로 교체(프론트 무수정).
- 검색 URL 실측 확인: 쿠팡·컬리(`?sword=`)·이마트몰(`emart.ssg.com/search.ssg?query=`)·네이버쇼핑 전부 정상.
- 네이버 참고가는 `NAVER_CLIENT_ID/SECRET` 있을 때만(없으면 딥링크만, 우아하게 degrade).

**중립성 해자는 딥링크만으로도 성립한다:** 쿠팡 앱은 "다른 4개 마트로 보내주는" 기능을 죽어도 못 넣는다.

## 5. 로드맵

- **지금:** 4개 마켓 딥링크(쿠팡·컬리·SSG·네이버). 쿠팡·네이버는 배포 후 제휴 신청해 수익 연결.
- **컬리·SSG 제휴 문의** → 되면 `MARKETS[].affiliate=true`로 켜기.
- **트래픽 쌓이면** 링크프라이스로 11번가·G마켓 추가.
- **네이버 검색 API 키** 넣으면 참고 최저가 자동 표시.
- **쿠팡파트너스는 배포 URL 있어야 신청 가능**(Phase 5).
